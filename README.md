# 💎 Gold Store Management System

A web-based Gold Store Management System developed using **ASP.NET Web Forms** and **C#** to simplify jewelry store management operations.

## 📌 Overview

This project provides an administrative dashboard for managing different aspects of a gold jewelry store, including products, categories, users, and system logs.

It was developed as part of my university studies to improve my skills in web application development using ASP.NET.

---

## ✨ Features

- 🔐 Secure Login System
- 📊 Admin Dashboard
- 👤 User Management
- 💍 Product Management
- 📂 Category Management
- ⚙️ Parameter Management
- 📝 System Log Management

---

## 🛠️ Technologies Used

- ASP.NET Web Forms
- C#
- .NET Framework
- SQL Server
- HTML5
- CSS3
- Bootstrap

1. Clone the repository


3. Restore NuGet packages.

4. Configure the SQL Server database.

5. Run the project.
